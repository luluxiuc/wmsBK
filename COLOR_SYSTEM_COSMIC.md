# wms 宇宙蓝色彩系统设计规范

## 设计理念

传达"爱思考"、"忧郁"、"蓝天"和"宇宙的深邃"等核心意境，通过从蓝天到深空的色彩渐变，营造宁静而深邃的视觉体验。

## 色彩心理学

- **低饱和度蓝色系**: 传达思考、忧郁、宁静的氛围
- **深紫色与靛蓝色**: 表现宇宙的神秘与深邃
- **明亮天蓝色**: 作为希望与交互的亮点

## 色彩规范

### 主色调 - 宇宙蓝渐变

| 色彩名称 | 色值 | 应用场景 |
|---------|------|---------|
| **深空黑** | `#0B0F19` | 最深背景，模拟宇宙深处 |
| **午夜蓝** | `#0F172A` | 主背景色，页面底色 |
| **深宇宙蓝** | `#1E293B` | 卡片背景，次级容器 |
| **星云蓝** | `#334155` | 边框、分隔线 |
| **天空蓝** | `#60A5FA` | 主要强调色，按钮 |
| **浅天蓝** | `#93C5FD` | 悬停状态，高亮 |
| **星光白** | `#F8FAFC` | 主文字颜色 |
| **月光灰** | `#CBD5E1` | 次级文字，描述 |

### 辅助色 - 宇宙紫/靛青

| 色彩名称 | 色值 | 应用场景 |
|---------|------|---------|
| **深紫** | `#4C1D95` | 渐变终点，神秘元素 |
| **靛蓝** | `#3730A3` | 渐变中间色 |
| **紫罗兰** | `#8B5CF6` | 装饰元素，特殊强调 |

### 渐变方案

```css
/* 主背景渐变 - 从深空到午夜 */
--gradient-cosmos: linear-gradient(180deg, #0F172A 0%, #0B0F19 100%);

/* 英雄区渐变 - 蓝天到深空 */
--gradient-sky: linear-gradient(180deg, #1E3A8A 0%, #0F172A 50%, #0B0F19 100%);

/* 卡片渐变 - 微妙的深度 */
--gradient-card: linear-gradient(145deg, #1E293B 0%, #0F172A 100%);

/* 强调渐变 - 天空到靛蓝 */
--gradient-accent: linear-gradient(135deg, #60A5FA 0%, #3730A3 100%);
```

## 应用场景

- **页面主背景**: `#0F172A` (午夜蓝)

- **边框颜色**: `#334155` (星云蓝)

## 设计原则

1. **层次清晰**: 通过色彩的深浅变化建立视觉层次
2. **过渡自然**: 所有渐变都要平滑自然，模拟真实的天空
3. **对比适度**: 确保文字与背景有足够的对比度，保证可读性
4. **统一性**: 所有页面遵循相同的色彩规范
5. **情感传达**: 通过冷色调营造思考与宁静的氛围

## 响应式适配

- 移动端保持相同的色彩方案
- 适当调整渐变的强度以适应小屏幕
- 确保在暗光环境下也能舒适阅读
