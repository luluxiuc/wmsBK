# wmsBK 设计系统 v2.0

## 设计灵感
基于 Linktree 现代、活力、高对比度的设计语言，打造年轻、专业、有活力的个人品牌形象。

## 色彩系统

### 主色调
- **青柠绿 (Lime)**: `#D4F938` - 主强调色，CTA按钮，高亮元素
- **深森林绿 (Forest)**: `#1B4D3E` - 深色背景，主要文字
- **墨绿 (Dark Green)**: `#243C2E` - 次级深色，卡片背景

### 中性色
- **纯白**: `#FFFFFF` - 主背景
- **浅灰**: `#F5F5F5` - 次级背景
- **中灰**: `#9CA3AF` - 辅助文字
- **深灰**: `#374151` - 正文文字
- **纯黑**: `#1A1A1A` - 标题文字

### 使用规则
- 浅色模式: 白底 + 黑字 + 青柠绿强调
- 深色模式: 深森林绿底 + 白字 + 青柠绿强调

## 字体系统

### 字体家族
- **标题**: Inter, system-ui, sans-serif (粗体 700-900)
- **正文**: Inter, system-ui, sans-serif (常规 400-500)

## 组件规范

### 按钮

**主按钮 (Primary)**
- 背景: `#D4F938`
- 文字: `#1B4D3E`
- 圆角: 9999px (胶囊形)
- 内边距: 16px 32px
- 字重: 600
- Hover: 亮度降低 5%

**次按钮 (Secondary)**
- 背景: `#1B4D3E`
- 文字: `#FFFFFF`
- 圆角: 9999px
- 内边距: 16px 32px
- Hover: 背景变深

**幽灵按钮 (Ghost)**
- 背景: transparent
- 边框: 2px solid `#1B4D3E`
- 文字: `#1B4D3E`
- 圆角: 9999px

### 卡片

**项目卡片**
- 背景: `#FFFFFF` 或 `#243C2E` (深色模式)
- 圆角: 16px
- 阴影: 0 4px 20px rgba(0,0,0,0.08)
- 内边距: 24px
- Hover: 阴影加深，轻微上移

**Bento 卡片**
- 背景: `#F5F5F5` 或 `#243C2E`
- 圆角: 24px
- 内边距: 32px

### 导航栏

**悬浮导航**
- 背景: `#FFFFFF`
- 圆角: 9999px (胶囊形容器)
- 阴影: 0 4px 20px rgba(0,0,0,0.1)
- 内边距: 12px 24px
- 位置: 固定顶部，居中

### 输入框

**文本输入**
- 背景: `#FFFFFF`
- 边框: 2px solid `#E5E7EB`
- 圆角: 12px
- 内边距: 16px 20px
- Focus: 边框变为 `#D4F938`

## 动画规范

### 过渡时间
- 快速: 150ms (按钮状态)
- 正常: 300ms (卡片悬停)
- 慢速: 500ms (页面过渡)

### 缓动函数
- 默认: cubic-bezier(0.4, 0, 0.2, 1)
- 弹性: cubic-bezier(0.34, 1.56, 0.64, 1)
