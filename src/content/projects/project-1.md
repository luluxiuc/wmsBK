# 个人作品集网站

这是我使用 Astro 框架构建的个人作品集网站，展示了我在前端开发和产品设计方面的能力。

## 主要特性

- **极速性能**: 使用 Astro 的 Islands 架构，实现极快的页面加载速度
- **响应式设计**: 完美适配桌面、平板和移动设备
- **深色模式**: 支持系统偏好和手动切换
- **自动化部署**: 通过 GitHub Actions 实现 push 即部署
- **内容驱动**: 使用 Markdown 管理内容，轻松更新

## 技术栈

- Astro 5.0
- Tailwind CSS
- TypeScript
- GitHub Actions

## 项目亮点

这个项目不仅是一个展示平台，更是我对现代 Web 开发最佳实践的实践。从性能优化到用户体验，每个细节都经过精心打磨。
