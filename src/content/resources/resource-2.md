# Tailwind CSS 组件库

Tailwind UI 是 Tailwind CSS 的官方组件库，提供精心设计的组件和模板。

## 主要内容

- **页面模板**: 完整的页面设计模板
- **应用组件**: 常见的应用界面组件
- **营销组件**: 营销页面常用组件
- **电商组件**: 电商网站专用组件

## 使用建议

虽然部分资源需要付费，但免费资源已经足够丰富。建议先使用免费资源熟悉 Tailwind 的开发模式。
